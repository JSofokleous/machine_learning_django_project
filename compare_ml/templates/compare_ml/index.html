This project attempts to test different machine learning models on a real dataset on rent prices in New York.

Each 'data point' is a property, with features such as size, number of bedrooms and distance to underground. Every property has an associated label: within the stated budget OR not with the stated budget.

Given an input sample with the features of your desired propety, each of the machine learning models will predict if this property is within your budget, based on the features and labels of the properties in the dataset.
